# TaskFlow

TaskFlow is an interactive task and project management app with a Kanban board, analytics dashboard, and a flexible team workflow system.

![TaskFlow Screenshot](./public/screenshot.png)

## Demo & Repository

- **Demo:** [task-flow231.vercel.app](https://task-flow231.vercel.app)
- **Repo:** [github.com/shrileon/TaskFlow](https://github.com/shrileon/TaskFlow)

## Features

- **Dashboard** — project activity overview: task stats, status donut chart, project bar chart, recent tasks
- **Kanban Board** — drag & drop tasks between columns (Backlog → In Progress → Review → Done)
- **Project Management** — create and delete projects, track progress, manage members
- **Task Details** — edit status, priority, assignee; checklist with progress bar; comments; labels and due dates with overdue indication
- **Task Creation** — modal with title, description, status, priority, assignee, and due date
- **Dark / Light Theme** — toggle in settings
- **Search** — instant task filtering by title and description
- **Responsive Design** — works on mobile, tablet, and desktop
- **Animations** — smooth transitions and micro-interactions

## Tech Stack

React 19 · TypeScript · Vite · Tailwind CSS · Zustand · React Router · @dnd-kit · Framer Motion

## Getting Started

**Prerequisites:** Node.js ≥ 18, npm ≥ 9

```bash
git clone https://github.com/shrileon/TaskFlow.git
cd TaskFlow
npm install
npm run dev
```

App will be running at [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```

Output goes to the `dist/` directory.
